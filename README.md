## 📌 项目简介

**智历**是一款面向大学生和求职者的 AI 简历诊断工具。用户只需粘贴简历内容，系统便会调用大语言模型从**六个维度**进行全方位深度分析，生成可视化诊断报告，并提供具体的修改建议和优化示例。

本项目由 **西安电子科技大学 网络与信息安全学院** 学生团队开发，将 AI 技术与网络安全视角相结合，在提供简历优化建议的同时，帮助用户识别和防范简历中的隐私泄露风险。

## ✨ 核心功能

### 🎯 六维度评分体系

| 维度 | 说明 |
|------|------|
| 📋 **内容完整性** | 教育背景、实习/工作经历、项目经历、技能、自我评价等模块是否齐全 |
| ✍️ **表达专业度** | 是否使用STAR法则、是否量化成果、用词是否专业规范 |
| 📐 **排版规范性** | 格式是否统一、层次是否清晰、信息密度是否合理 |
| 🔒 **隐私安全性** | 是否暴露身份证号、家庭住址、银行卡号等敏感信息（网安特色） |
| 🎯 **ATS兼容性** | 关键词覆盖率分析，预估自动筛选系统通过率 |
| 💼 **行业适配度** | 针对目标行业（互联网/金融/咨询/公务员等）的定制化建议 |

### 📊 可视化诊断报告

- **雷达图**：六维度评分可视化对比
- **问题诊断**：按严重/中等/轻微三级分类，附具体修改建议
- **隐私风险检测**：自动识别简历中的敏感信息并给出风险评级
- **ATS关键词分析**：展示匹配与缺失的行业关键词，预估通过率
- **STAR法则检测**：逐段检查经历描述是否符合STAR法则
- **优化前后对比**：关键段落的优化前后对比，直观展示改进方向

### 📁 数据管理

- **历史记录**：自动保存每次诊断结果，追踪简历改进趋势
- **行业关键词库**：内置互联网/金融/咨询/公务员四大行业核心关键词
- **报告导出**：一键导出 TXT 格式诊断报告
- **工作台仪表盘**：累计诊断次数、平均分、隐私风险统计等数据概览

## 🚀 快速开始

### 方式一：浏览器直接使用

1. 下载 `index.html`
2. 双击用浏览器打开
3. 配置 API Key 或选择演示模式
4. 粘贴简历 → 开始诊断

### 方式二：桌面程序

**环境要求**：Windows 10/11 + Python 3.8+

```bash
# 克隆项目
git clone https://github.com/你的用户名/ZhiLi.git
cd ZhiLi

# 安装依赖
pip install pywebview

# 启动程序
python app.py
```

或直接双击 `start.bat` 一键启动。

### 方式三：安装包（推荐）

下载 Releases 页面的 `ZhiLi_v2_Setup.exe`，双击安装即可使用，包含桌面快捷方式和开始菜单项。

## 🔒 隐私与安全

- API Key 仅保存在用户本地浏览器的 localStorage 中，不会上传至任何服务器
- 简历内容直接发送至用户配置的 AI API，本项目不做任何数据收集和存储
- 隐私检测功能检测范围：身份证号、详细家庭住址、银行卡号、过于详细的个人信息等
- 所有数据处理均在用户本地完成

## 📌 项目背景与意义

### 解决的问题

- **简历质量参差不齐**：大学生缺乏专业简历指导资源，简历普遍存在表达空泛、缺乏量化数据等问题
- **隐私保护意识薄弱**：求职者经常在简历中暴露身份证号、详细家庭住址等敏感信息而不自知
- **ATS系统筛选淘汰率高**：简历关键词覆盖不足导致被企业招聘系统自动过滤
- **传统修改效率低**：人工简历修改成本高、周期长、质量不稳定

### 创新点

- **六维度评分体系**：将隐私安全和ATS兼容性纳入简历评分维度
- **网安特色隐私检测**：融入网络安全视角，自动识别简历中的敏感信息泄露风险
- **STAR法则自动化检测**：逐段分析经历描述，给出针对性改写建议
- **行业关键词库**：内置多行业关键词，精准提升ATS通过率
- **零成本即时可用**：支持演示模式，无需任何费用即可体验完整功能

## 🔧 自行构建安装包

如需将项目打包为 Windows 安装程序：

1. 安装 [Inno Setup 6](https://jrsoftware.org/isdl.php)（免费）
2. 双击 `step1_build_exe.bat`（打包 exe，约1-3分钟）
3. 双击 `step2_make_installer.bat`（生成安装包）
4. 在 `installer_output/` 文件夹中获取 `ZhiLi_v2_Setup.exe`
