# ZhiLi v2.0 - AI Resume Analyzer

## Files
```
ZhiLi_v2/
├── start.bat       <- Double-click to run
├── build_exe.bat   <- Package as standalone exe
├── app.py          <- Python launcher
├── index.html      <- Frontend (all-in-one)
└── README.md
```

## Quick Start
1. Double-click `start.bat` (auto-installs dependencies)
2. Configure API or use Demo Mode
3. Paste resume → Analyze → View 6-dimension report

## Features
- 6-dimension scoring (completeness, expression, format, privacy, ATS, industry)
- Privacy leak detection (ID numbers, addresses, etc.)
- ATS keyword matching analysis
- STAR method checker
- Before/after optimization comparison
- Radar chart visualization
- History tracking & export
- Industry keyword database
